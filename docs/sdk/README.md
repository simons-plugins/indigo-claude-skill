## Using This Documentation

Each subdirectory contains markdown files that:
1. Link to relevant official documentation sections
2. Provide context for when to use specific features
3. Include code snippets and examples
4. Reference example plugins in this SDK

Navigate to specific topics based on what you're working on:
- Creating your first plugin? Start with [getting-started/](getting-started/)
- Need to understand device lifecycle? See [plugin-development/device-lifecycle.md](plugin-development/device-lifecycle.md)
- Looking up API methods? Check [api-reference/](api-reference/)
- Want to see how others did it? Browse [examples/](examples/)
